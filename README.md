# SuperX 广告发布系统

一个简洁而强大的广告发布和展示平台。

作者: Peter

## 功能特点

### 首页功能
- 响应式布局，自适应不同屏幕尺寸
- 广告图片自动适配显示
- 支持广告图片点击跳转外部链接
- 右上角快速跳转按钮，方便定位未固定广告
- 顶部导航栏包含快捷功能按钮
- 一键发布广告入口

### 广告发布功能
- 支持图片上传和预览
- 拖拽调整广告位置和大小
- 智能防止广告重叠
- 支持为广告添加外部链接
- 固定位置功能
- 实时预览效果
- 一键清空画布
- 快速返回首页
